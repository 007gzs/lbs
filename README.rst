安装
********

目前 Lbs 支持的 Python 环境有 2.7, 3.4, 3.5, 3.6, 3.7 和 pypy。

为了简化安装过程，推荐使用 pip 进行安装

.. code-block:: bash

    pip install lbs

升级 lbs 到新版本::

    pip install -U lbs

如果需要安装 GitHub 上的最新代码::

    pip install https://github.com/007gzs/lbs/archive/master.zip
